Project 02

Rinata Restaurant AI Agent Suite

Python · Claude API · Gmail API · Google Sheets

6 Builds Shipped
6–10 hrs Weekly labor recovered
5 Email categories auto-triaged
<$1 API cost per full run
100% Idempotent & auditable

A multi-phase AI agent suite built and deployed for a real operating restaurant (Rinata, South Minneapolis). Each build targets a specific manual workflow.

Build 1–2: Google Reviews scraper — categorizes every review by sentiment, theme, and staff mention into a structured Excel report. Build 3–4: Supplier email triage — connects to Gmail, classifies every supplier email (invoice, delivery, price change, dispute), flags urgencies, and outputs a color-coded triage workbook. Build 5–6: Vendor invoice reconciliation — extracts line items from PDFs, matches against master price list, surfaces discrepancies.

Governing principle: deterministic Python does the work; Claude handles only judgment calls that require language understanding.

Project 03

Personal Email Intelligence System

Python · Claude API · Gmail API

In Development
4 Priority tiers (Now/Today/Later/Ignore)
0 Emails auto-sent (human in loop)
1 Daily digest generated
100% Reversible actions only

An AI agent for the personal inbox: triages incoming email by priority, drafts replies in the user's voice (saved as Gmail drafts — never auto-sent), applies reversible labels to organize noise, and produces a morning digest.

Built on four non-negotiable principles: human approves every outgoing message, reversible-only actions (labels, never deletes), dry-run mode by default, and idempotent scheduling so it's safe to run on a cron job.

Architecture mirrors the Rinata suite: deterministic Python orchestrates, Claude Haiku handles classification and drafting at minimal cost.
